## Установка и запуск
Для запуска Docker контейнера используйте следующие команды:
```
git clone https://github.com/XeeRooX/minitodo.git
cd ./minitodo
docker compose up
```
Далее в браузере перейдите по следующему адресу:
``` http://localhost:5000 ```
